# DrGeo

A repository for porting of DrGeo to Pharo 7

http://www.drgeo.eu/

### How to load

```
Metacello new
  baseline: 'DrGeo';
  repository: 'github://pavel-krivanek/DrGeo/src';
  load.
```

### How to run

```
DrGDefault beWorkstation.
DrGeo newFullScreen
```
